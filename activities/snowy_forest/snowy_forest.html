Create custom trees and add them to the forest!

SNOWY FOREST ANIMATION





DESCRIPTION
Difficulty level: Intermediate (Requires experience making Elm shapes)

Each student uses their Elm graphics skills to design their own tree. The trees are then used to create a unique, animated forest that can be showcased on a school or class website, or printed out to put up in the classroom!


WHAT TO DO
Click on the "Tree with a House - Code" link below and copy ALL the code (press Ctrl A then Ctrl C) and paste the code (Ctrl V) into http://elm-lang.org/try
